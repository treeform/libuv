# nimuv
> nimuv is libuv dynamic binding for nim.
### doc
see official libuv api document [libuv](http://docs.libuv.org/en/v1.x/api.html)
### require
- libuv (test 1.18)
- nim (test 0.17.2)
### usage
see uvtest.nim
warning: change your install libuv path in nimuv.nim. (default is /usr/local/libuv.so)
